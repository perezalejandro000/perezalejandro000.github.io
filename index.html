Mis proyectos


Cerradura inteligente

Este proyecto se desarrolló para implementar un sistema de seguridad electrónico sin necesidad de llaves, aplicable a cajas fuertes, puertas o portones, con el objetivo de evitar la pérdida u olvido de llaves.

Pneumoscan

El proyecto utiliza machine learning y deep learning para analizar imágenes de rayos X y mejorar el diagnóstico de neumonía, acelerando el proceso y aumentando la precisión, lo que optimiza la atención médica y reduce los tiempos de respuesta.

P.P.G

Este proyecto destaca la complejidad y riqueza de las reglas gramaticales, y busca ser útil para mejorar habilidades lingüísticas y comprender la gramática. Además, intenta cambiar esa situación mediante un juego educativo.
